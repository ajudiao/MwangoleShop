O comando 
    npm i -D @types/leaflet 

serve para instalar as definições de tipos do TypeScript para a biblioteca Leaflet, salvando-as como uma dependência de desenvolvimento (-D).Como o Leaflet original foi escrito em JavaScript puro, o TypeScript não consegue entender nativamente quais são as funções, objetos e propriedades que ele possui. Este pacote resolve esse problema.O que acontece na prática ao executar o comando:Ativa o Autocompletar (IntelliSense): O seu editor de código (como o VS Code) passará a sugerir os métodos corretos e os parâmetros necessários assim que começar a digitar comandos do Leaflet (ex: .map(), .marker()).Previne Erros de Compilação: O compilador do TypeScript deixará de acusar o erro comum Cannot find module 'leaflet' or its corresponding type declarations.Segurança no Código: Garante que você está a passar os tipos de dados corretos (como coordenadas em formato numérico e não em texto), capturando erros antes mesmo do código ser executado.O significado da flag -D (ou --save-dev)Significa que este pacote só é necessário enquanto você está a desenvolver o projeto (no seu computador ou no servidor de build). Quando a aplicação for compilada e enviada para produção (para os utilizadores finais), estas definições de tipo são descartadas, mantendo o tamanho final do projeto mais leve.
